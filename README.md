# RF toolbox
Streamlit app providing various interactive RF tools

- Touchstone (s-parameters) plotter
- Simple RF matching using an interactive Smith chart
- Analog filter designer
- FIR filter designer
- Wireless link budget calculator

Python Requirements:

- python 12
- Streamlit
- Plotly
- Scikit-rf
- numpy
- scipy

Run it locally by typing "streamlit run app.py" in a command prompt
